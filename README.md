# SkillMatch Service Platform

## Project Overview

SkillMatch is a MERN-based service marketplace platform developed as a university group project.

The platform connects service seekers with service providers through service requests, booking management, subscriptions, secure authentication, and role-based dashboards.

## Technologies Used

* React.js
* Node.js
* Express.js
* MongoDB
* JWT Authentication
* Tailwind CSS
* Stripe Payment Integration

## Features

* Service Provider Dashboard
* Service Seeker Dashboard
* Service Request Management
* Subscription Services
* Booking Management
* Role-Based Access Control
* Secure Authentication
